Жадный алгоритм:

- для каждой пары (video, cache) считаем выигрышь, который даст размещение видео в этом кэше
    - для каждого запроса (video, endpoint, count) итерируемся по кешам, с которыми связан endpoint
    - обновляем полезность video для кеша как:
        utility[video, cache] += max(0, dcLatency - cacheLatency) * count
- для каждого кеша сортируем видое по убыванию полезности
- кладём в кеш видео с максимальной полезностью пока он не заполненится
